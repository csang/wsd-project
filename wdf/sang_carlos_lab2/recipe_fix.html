Recipe: HTML Greatness

Ingredients

  • HTML Elements
  • Attributes to provide additional information
  • Open and close body tags to contain content
  • Content

Directions

  1. Code should not just be stirred together; it should be nested and well-formed.
  2. Use closing tags for properly formed elements
  3. Add entities when necessary for flavor
  4. Cook in the browser for 1 second
  5. Serve and enjoy!

Note: For extra goodness, add in a DOCTYPE and validate.
