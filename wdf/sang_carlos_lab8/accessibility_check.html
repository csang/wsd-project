www.stephanrizon.com

This is the personal website of Stephan Rizon. It contains its biography, videos, photos, contact information and more.

Passed Cynthia Says verification

WCAG:

  • Must add the language code.
  • Must use title attribute or add a text for the anchor elements.
  • The id attribute must be unique.

mars-one.com

This is a site that talks about the plan of living in Mars from 2023. It contains their mission, the team, news about mars one and more.

Cynthia Says:

  • Rule: 1.1.2 - All INPUT elements are required to contain the alt attribute or use a LABEL.
  • Failure - INPUT Element, of Type TEXT, found at Line: 88, Column: 2.
  • Failure - INPUT Element, of Type EMAIL, found at Line: 266, Column: 2.

WCAG:

  • Same error as Cynthia says.
  • The contrast between the color of text and its background for the element is not sufficient to meet WCAG2.0 Level AA. Use better contrasted colors.
  • i (italic) element used. Use em instead.
  • Anchor contains no text. Add text to the a element or the title attribute of the a element or, if an image is used within the anchor, add Alt text to the image.
  • Label text is empty. Add text to the label element.

panic.com/coda

This is the site of a plain text editor used to make websites called Coda. It contains the link to download it, a support link, plug-ins, benefits of the new Coda, and more.

Passed Cynthia Says verification

WCAG:

  • The contrast between the colour of text and its background for the element is not sufficient to meet WCAG2.0 Level AA. Use better contrasted colors.
  • Header nesting - header following h1 is incorrect. Modify the header levels so only an h1 or h2 follows h1.

crayonstoconcept.com

This is a showcase site of the SPU senior design students. It contains images of the students, a link to their works, a link to their exhibition information and their program.

Cynthia Says:

  • All images must have the alt or the longdesc attribute. Add the alt attribute.

WCAG:

  • Same error as Cynthia Says. Add the alt attribute.
